# College Discovery Platform — Frontend MVP

A modern frontend application built to help users discover, filter, and compare colleges. This project was developed as an MVP for a Frontend Developer internship application.

## 🚀 Tech Stack

* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Library:** React
* **Language:** TypeScript / JavaScript
* **Styling:** CSS Modules / Tailwind CSS (Update based on your exact usage)
* **Deployment:** Vercel

## ✨ Key Features

* **Listing Page:** Browse a directory of available colleges.
* **Detail View:** View in-depth information about specific colleges.
* **Compare Feature:** Select multiple colleges to compare their attributes side-by-side.
* **Sign-In Flow:** User authentication interface.

## 🛠️ Getting Started Locally

To run this project on your local machine:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/Biraj682/college-discovery-frontend.git](https://github.com/Biraj682/college-discovery-frontend.git)

 2. Navigate to the project directory:

Bash
cd college-discovery-frontend

3. Install dependencies:

Bash
npm install

4. Run the development server:

Bash
npm run dev 

5. Open your browser:
Navigate to http://localhost:3000 to see the application.
