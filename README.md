# College Discovery — MVP scaffold

This scaffold contains a Vite + React + TypeScript starter with Tailwind CSS, React Query, Zustand, and example pages (Listing, Detail, Compare, SignIn).

To create and run locally:
1. Ensure Node.js is installed (>=18)
2. Run `node extract-scaffold.js` to materialize src/ files
3. Run `npm install`
4. Run `npm run dev`

Note: Storybook and Cypress are configured in package.json; run after installing dependencies.
